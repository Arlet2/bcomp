О модификации
Добавлена возможность загрузки микропрограммы из файла (mp.txt).
Сейчас в mp.txt (в папке dist) загружены микрокоманды стандартного интерпретатора БЭВМ.
Рабочий jar-файл (также лежит в папке dist) - mabcomp.jar. В одной папке с ним обязательно должен находится mp.txt.

При невозможности загрузки из файла, будет загружен стандартный интерпретатор БЭВМ.
Рекомендуется проверять микропрограмму на ошибки путём загрузки БЭВМ через консоль (java -jar -Dmode=cli mabcomp.jar).
