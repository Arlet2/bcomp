О модификации
Добавлена возможность загрузки микропрограммы из файла (mp.txt), что даёт возможность пользоваться всеми возможностями UI с изменненой микропрограммой БЭВМ.
В mp.txt загружены микрокоманды стандартного интерпретатора БЭВМ, но его можно изменить (в рамках 7 лабы).
Рабочий jar-файл - mabcomp.jar. При запуске в одной папке с ним обязательно должен находиться mp.txt.

При невозможности загрузки из файла, будет загружен стандартный интерпретатор БЭВМ.
Рекомендуется проверять микропрограмму на ошибки путём загрузки БЭВМ через консоль (java -jar -Dmode=cli mabcomp.jar).

Формат указания микрокоманд: MICROCODE_HEX LABEL (пример можно увидеть в файле mp.txt)

Полностью новый интерпретатор в данный момент написать невозможно.
Метка RESERVED обязательна, её не стоит изменять.
